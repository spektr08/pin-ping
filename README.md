# Pin-Pong React&Redux Client
## How to start

First you need to run api server https://github.com/spektr08/ping-pong-server

Installation
------------
1. - ```npm isntall``` 
2. - change url to api in webpack.config.js 
```
	config: JSON.stringify({
            apiUrl: 'http://192.168.99.100'
        })
```
3. - ```npm start```
