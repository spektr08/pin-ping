# pin-ping 
## How to start

First youu need to run api server https://github.com/spektr08/ping-ping-server

Installation
------------
1. - ```npm isntall``` 
2. - change url to api in webpack.config.js 
```
	config: JSON.stringify({
            apiUrl: 'http://192.168.99.100'
        })
```
3. - ```npm start```
